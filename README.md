# Description


## Install

```bash
npm i
```
## Environment template

```python
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_anon_key
```
## Run development

```bash
npm run dev
```

## Run tests
I use Vitest for Unit Testing

```bash
npm test
```

## Build the project

```bash
npm run build
```
